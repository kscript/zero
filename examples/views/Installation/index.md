## 安装


### npm 安装
```
npm i -D @zero-ui/vue
```
``` javascript 
// 在项目入口文件中引入并使用

// 组件库
import zeroUI from '@zero-ui/vue'
// 样式文件
import '@zero-ui/vue/lib/zero.css'

```
